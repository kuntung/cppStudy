### 程序功能
1. 开始演讲比赛：完成整届比赛的流程，每个比赛阶段需要给用户一个提示，用户按
   任意键后继续下一阶段
   
2. 查看往届记录：查看之前比赛前三名结果，每次比赛都会记录到文件中，文件用`.csv`
后缀名保存
   
3. 清空比赛记录：将文件中数据清空
4. 退出比赛程序：可以退出当前程序
