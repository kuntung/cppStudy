## `explicit`
通过将**构造函数**声明为`explicit`，可以阻止他们被用来执行**隐式类型转换**，尽管他们仍然可以进行**显式类型转换**

因此，除非允许构造函数被用于隐式类型转换，否则将其声明为`explicit`
